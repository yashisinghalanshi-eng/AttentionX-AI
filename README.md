# 🎬 AttentionX AI

## 🚀 Problem
Long videos are time-consuming. People want short, engaging clips.

## 💡 Solution
AttentionX AI automatically:
- Extracts audio from videos
- Converts speech to text using AI
- Detects important moments
- Generates short, vertical clips with captions

## 🔥 Features
- Video Upload
- AI Transcription (Whisper)
- Important Moment Detection
- Auto Clip Generation
- Vertical (Reels Ready)
- Caption Overlay

## 🛠️ Tech Stack
- Python
- Streamlit
- MoviePy
- OpenAI Whisper

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
